Variables and Data Types

In JavaScript, variables are used to store data, and the language supports several data types, like:

  • strings;
  • numbers;
  • booleans;
  • objects;
  • arrays, and more.

Variables can be declared using:

  • var;
  • let;
  • const;

For example, you can declare a string using let like this:

let name = "John";

JavaScript's data types include primitives like string, number, and boolean, as well as complex types like object and array. It’s important to note how JavaScript handles type coercion, automatically converting between types when necessary. For instance, "5" + 5 results in the string "55" due to type coercion.

Functions and Scope

Functions in JavaScript are blocks of reusable code designed to perform specific tasks. You can declare a function using the function keyword, as shown here:

function greet() { console.log("Hello!"); }

Functions can also take parameters and return values, allowing them to be highly flexible. Understanding scope is crucial when working with functions. For example, variables declared inside a function using let are only accessible within that function, as in:


let x = 10; function test() { let y = 20; }

Arrow functions provide a more concise way to write functions and have different behavior regarding this. Here’s an example:

const add = (a, b) => a + b;
Control Structures and Loops

Control structures like if, else, and switch allow you to control the flow of your code based on conditions. For example, the following code checks if a number is positive:

if (number > 0) { console.log("Positive"); }

Loops like for, while, and do...while let you repeat code blocks as needed. A for loop, for instance, iterates over a sequence of numbers:

for(let i = 0; i<5; i++) {console.log(i);}

You can use break to exit a loop early or continue to skip to the next iteration.

Objects and Arrays

Objects and arrays are fundamental structures in JavaScript for storing collections of data. An object stores data as key-value pairs, like this: let person = { name: "Alice", age: 30 };. You can access the properties of an object using dot notation or bracket notation.

Arrays are ordered lists of items, which can be accessed by index. For example, let fruits = ["apple", "banana", "cherry"]; creates an array of fruits. JavaScript provides various methods for manipulating arrays, such as fruits.push("orange");, which adds an item to the end of the array.

Asynchronous JavaScript

Asynchronous programming is essential in JavaScript for handling operations that may take time to complete, such as fetching data from an API. One way to handle asynchronous tasks is by using callbacks, where a function is passed as an argument to be executed later, like this: setTimeout(() => { console.log("Done!"); }, 1000);.

Promises offer a cleaner way to work with asynchronous operations, with methods like promise.then().catch().finally();. The async/await syntax provides an even more readable approach, allowing you to write asynchronous code that looks synchronous. For instance, async function fetchData() { let response = await fetch(url); } waits for the fetch operation to complete before moving on.
